# 网络安全协议基础笔记

本节整理了常见网络安全协议及其应用层次、端口和用途，便于复习和考试掌握重点。

---

## 一、常见网络安全协议及应用层次

| 协议 | 主要用途 | 层次 | 备注/端口 |
|------|----------|------|-----------|
| PGP (Pretty Good Privacy) | 邮件加密与签名 | 应用层 | 与POP3、SMTP配合使用，保证邮件内容安全 |
| SSL (Secure Sockets Layer) | Web传输加密 | 应用层/传输层 | 与HTTP结合形成HTTPS，防止明文传输，默认端口443 |
| TLS (Transport Layer Security) | 传输层安全 | 传输层 | TLS是SSL升级版，主要用于传输层加密 |
| SET (Secure Electronic Transaction) | 电子商务交易安全 | 应用层 | 中间的E代表电子（Electronic） |
| IPSEC (Internet Protocol Security) | 网络层数据安全 | 网络层 | 基于IP层，保障数据报文完整性和加密 |

---

## 二、协议特点与记忆技巧

1. **PGP → 邮件安全**  
   - 可与POP3、SMTP配合  
   - 记忆技巧：两个P → 邮件协议（POP3）

2. **SSL/TLS → Web安全**  
   - HTTP明文 → HTTP + SSL = HTTPS  
   - 默认端口：HTTP → 80，HTTPS → 443  
   - TLS是SSL升级版，强调传输层安全  

3. **SET → 电子商务**  
   - 中间的E = Electronic → 电子  
   - 用于保证交易安全和支付信息加密  

4. **IPSEC → 网络层加密**  
   - 基于IP协议层  
   - 提供数据加密、认证和完整性保护  

---

## 三、典型考点

1. **HTTPS 基于 SSL/TLS**  
   - HTTP内容加密 → HTTPS  
   - 默认端口：443  

2. **电子邮箱安全**  
   - PGP用于邮件加密和签名  
   - 与MM1无关（MM1用于多媒体类型标识，不保证邮箱安全）  

3. **协议与层次对应表**  
| 协议 | 层次 | 用途 |
|------|------|------|
| PGP | 应用层 | 邮件加密 |
| SSL/TLS | 传输/应用层 | Web安全 |
| SET | 应用层 | 电子商务交易 |
| IPSEC | 网络层 | 数据报文加密 |

---

## 四、记忆小技巧

- **PGP** → 邮件（P开头，和POP3对应）
- **SSL/TLS** → Web安全 → HTTPS → 端口443
- **SET** → 电子商务 → 中间E = Electronic
- **IPSEC** → IP网络层 → 网络数据安全

---

## 五、典型真题解析

1. **HTTP vs HTTPS**  
   - HTTPS比HTTP更安全，因为基于SSL/TLS加密  
   - 端口：HTTPS → 443  

2. **电子邮箱安全相关协议**  
   - 正确：PGP  
   - 无关：MM1（多媒体类型标识协议）

3. **协议层次判断**  
   - 应用层：PGP、SET、HTTP/HTTPS  
   - 传输层：TLS  
   - 网络层：IPSEC

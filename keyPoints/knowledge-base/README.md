### 📖 文档标准格式

每个知识点文档严格遵循以下结构：

```markdown

## 一、考试重要性
- 考察频率：高/中/低
- 出现题型：选择题/案例题
- 是否必背：是/否

## 二、一句话结论（考试版）
> 一句话说清考点本质（可直接背）

## 三、核心考点（只保留考试要的）
- 要点1
- 要点2
- 要点3

## 四、典型真题考法
- 常见问法1
- 常见问法2

## 五、易错点 & 命题陷阱
- 易错点1
- 易错点2

## 六、记忆技巧 / 口诀
- 口诀 / 对比 / 表格

## 七、（可选）例题
【题目】
【正确答案】
【解析】
```

### 🔍 使用说明

1. **快速检索**：通过知识点编号快速定位
2. **考前复习**：重点关注"必背"标记的内容
3. **刷题练习**：结合例题巩固理解
4. **错题复盘**：查找对应易错点和命题陷阱

1. **优先级**：软件工程 > 面向对象 > 数据库 > 其他

# 软考中级软件设计师 - 知识点检索平台

## 🎯 项目简介

这是一个专为**软考中级软件设计师**考试打造的知识点文档库和在线检索平台，遵循"用最少的内容，拿最多的分"的核心理念。

## 📁 项目结构

```
├── index.html              # 主页面
├── styles.css              # 样式文件
├── app.js                   # 核心逻辑
├── knowledge-base/          # 知识点文档库
│   ├── README.md           # 文档库说明
│   ├── SE001-软件生命周期模型.md
│   ├── OO001-面向对象基本概念.md
│   ├── DB001-关系数据库理论.md
│   └── DS001-线性数据结构.md
└── README.md               # 项目说明
```

## 🚀 快速开始

### 方法一：直接打开（推荐）
1. 下载所有文件到本地目录
2. 双击 `index.html` 文件即可在浏览器中打开

### 方法二：本地服务器
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## ✨ 功能特性

### 📚 知识点文档库
- **标准化格式**：每个知识点严格按照考试导向的7段式结构编写
- **考试重点突出**：明确标注考察频率、题型、是否必背
- **易错点总结**：汇总命题陷阱和常见错误
- **记忆技巧**：提供口诀和记忆方法

### 🔍 智能检索平台
- **全文搜索**：支持标题、内容、关键词搜索
- **多维筛选**：按分类、重要性、题型筛选
- **双视图模式**：列表视图和卡片视图
- **详情查看**：点击知识点查看完整内容
- **统计信息**：实时显示知识点统计

### 📱 响应式设计
- 完美适配桌面端、平板、手机
- 现代化UI设计，操作流畅

## 📝 文档标准格式

每个知识点文档包含以下7个部分：

1. **考试重要性** - 频率、题型、是否必背
2. **一句话结论** - 核心要点总结
3. **核心考点** - 考试重点内容
4. **典型真题考法** - 常见出题方式
5. **易错点 & 命题陷阱** - 避免失分
6. **记忆技巧 / 口诀** - 高效记忆
7. **例题** - 真题练习

## 🎯 使用建议

### 备考阶段
1. **基础学习**：按分类系统学习，重点关注⭐⭐⭐⭐⭐内容
2. **强化练习**：结合例题和真题练习
3. **查漏补缺**：使用搜索功能快速定位薄弱知识点
4. **考前冲刺**：重点复习"必背"标记的内容

### 刷题阶段
1. 遇到不会的题目，通过搜索快速找到对应知识点
2. 查看"易错点"部分，避免重复犯错
3. 利用"记忆技巧"快速掌握要点

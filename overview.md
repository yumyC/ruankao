中级《软件设计师》学习规划要点（1.5～2 个月）
📌 1. 学习顺序（必看）

计算机基础（体系结构 + 操作系统 + 网络）

软件工程与 UML

数据结构与算法（最关键）

数据库与 SQL

信息安全基础

上午题全套知识点复习

下午题（程序设计）专项突破

📌 2. 核心知识点（按考试权重排序）
① 数据结构与算法（权重最高）

数组、链表、栈、队列

树与二叉树、B+树

图的 BFS/DFS

排序算法时间复杂度

查找（顺序、二分、散列表）

动态规划基础思路

👉 下午题编程一定会考！

② 软件工程 + UML（上午必考）

生命周期模型（瀑布、原型、敏捷、螺旋）

需求、概要设计、详细设计

UML：类图、序列图、用例图

CMMI、ISO9126、软件质量属性

③ 计算机网络 & 操作系统（容易得分）

TCP/IP 五层模型

HTTP、HTTPS 基础

进程/线程/死锁

页面置换算法（LRU、FIFO）

④ 数据库 & SQL

三大范式

事务（ACID）

连接、子查询

索引与优化简单原理

⑤ 信息安全

对称加密 / 非对称加密 / 哈希

防火墙、IDS/IPS 基本概念

常见攻击 SQL 注入、XSS

📌 3. 下午题（编程题）规划

语言任选（推荐：Python / C / Java）。
考点固定：

二叉树遍历

图搜索

动态规划

字符串处理

递归

排序与合并

查找
